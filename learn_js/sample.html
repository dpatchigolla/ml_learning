CSS basics:

  1. selector {property: value; property2:value2}
  2. .class {property: value}
  3. #id {property: value}
  1. Reference in head, with tag link. href has the name of file
  2. selector - what element to format/style (ex: h1)
    1. curly braces to enclose the style items
    2. first, property to change (ex: text-align )

class vs id

both class and id can be used to reference from stylesheet. class can be used for multiple items. id can be used only for one single item
