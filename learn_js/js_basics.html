JavaScript Basics

Syntax and basic concepts

JS is case-sensitive

  • Declaring Variable: var x = 3; Semicolon ends the statement
  • Expression. Declare another variable and then create expression on x and y
    • var y = 4;
    • var z = x + 2 * y;
  • Update existing variable: No need of prefix var. Ex: x = x+y;
  • Creating new 'objects': Using keyword new
    • var fgImg = new SimpleImage("image_path.jpg");
  • method vs function:
    • Method is invoked on an object, while function is not invoked on object
    • Method: var img = new SimpleImage("image.jpg");
    • Function: print(x); OR var y = square(4);
  • JS is dynamically typed language. We don't need to declare type of variables.
